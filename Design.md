# GharOS Design System

## Color Palette

| Token | Hex | Usage |
|-------|-----|-------|
| Primary | `#7A4520` | Buttons, active states, icons |
| Background | `#FBF6EF` | Screen backgrounds |
| Card | `#FFFDF9` | Card surfaces, modals |
| Accent | `#F0C98A` | Highlights, tags, progress bars |
| Text | `#2C1A0E` | Primary text |
| Border | `#E8D5B8` | Card borders, dividers |

## Typography

| Role | Font | Weight |
|------|------|--------|
| Headings | Lora (serif, Google Fonts) | 600–700 |
| Body | DM Sans | 400–500 |
| Captions / Labels | DM Sans | 400 |

## Spacing & Shape

- **Card border radius:** 14px
- **Button border radius:** 10px
- **Shadows:** None — use subtle `1px` borders (`#E8D5B8`) instead
- **Card elevation:** Border only, no drop shadow

## Aesthetic

Warm, earthy, Indian food aesthetic. Feels like a handwritten recipe notebook — not a cold productivity app.
Think: terracotta, turmeric, ghee, aged wood. Every screen should feel like it belongs in a home kitchen.

## Do / Don't

| Do | Don't |
|----|-------|
| Use warm off-whites and browns | Use pure white (`#FFFFFF`) backgrounds |
| Subtle golden accents sparingly | Overuse the accent color |
| Lora for section headings | Use Lora for body text or small labels |
| Soft borders to separate content | Harsh drop shadows |
| Round corners (14px cards) | Sharp corners or very large radius (>20px) |
